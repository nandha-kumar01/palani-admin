## Important Notes:

2. **Environment Types**: Set all to "Production, Preview, and Development" unless specified otherwise

## Steps to Add Variables:
1. Go to Vercel Dashboard
2. Select your project
3. Go to Settings → Environment Variables
4. Add each variable one by one
5. Set the environment to "Production, Preview, and Development"
6. Click "Save"
7. Redeploy your application
